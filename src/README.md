## 📌 Descripción del taller

Este taller tiene la implementación de varios ejercicios de recursividad
Los ejercicios incluyen operaciones matemáticas como:

- Factorial de un número
- Sumatoria de números
- Serie de fracciones (1 + 1/2 + 1/3 + ... + 1/n)
- Inversión de números
- Suma de dígitos
- Potencia de un número
- Máximo común divisor (MCD)
- División entera usando restas sucesivas
- Multiplicación usando sumas sucesivas
- Suma de arreglos
- Suma de matrices
- Serie de Fibonacci
- Función de Ackermann
